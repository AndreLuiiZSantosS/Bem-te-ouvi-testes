# CDU012. Arrematar

- **Ator principal**: Usuário logado(ouvinte ou músico).
- **Atores secundários**: Músico(criador do leilão).
- **Resumo**: Permite que o usuário autenticado(ouvinte ou músico) que fez o maior lance do leilão possa arrematar o item leiloado.
- **Pré-condição**: O leilão deve estar finalizado no sistema e o usuário deve ter feito o maior lance do leilão.
- **Pós-condição**: O item é arrematado pelo usuário e registra no sistema a ação.
