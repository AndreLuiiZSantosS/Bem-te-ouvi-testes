# CDU005. Doar

- **Ator principal**: Usuário logado.
- **Resumo**: Permite que o usuário autenticado(ouvinte ou músico) possa fazer uma doação para um músico.
- **Pré-condição**: O músico deve permitir a funcionalidade de receber doações.
- **Pós-condição**: O valor doado é transferido para a conta bancária do músico.
