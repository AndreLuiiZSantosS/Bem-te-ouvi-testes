# CDU013. Comentar

- **Ator principal**: Usuário logado(ouvinte ou músico).
- **Resumo**: Permite que o usuário autenticado(ouvinte ou músico) possa fazer comentários públicos em músicas disponíveis na plataforma.
- **Pré-condição**: O usuário deve estar autenticado no sistema e deve haver álbuns e músicos disponíveis no sistema.
- **Pós-condição**: O comentário é registrado e associado à música comentada.
