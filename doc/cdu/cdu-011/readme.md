# CDU011. Dar um lance

- **Ator principal**: Usuário logado(ouvinte ou músico).
- **Resumo**: Permite que os usuários autenticados(ouvinte ou músico) possam dar lances em leilões disponíveis.
- **Pré-condição**: O usuário deve estar autenticado(ouvinte ou músico) e deve haver leilões disponíveis.
- **Pós-condição**: O lance é registrado no sistema e mostrado no leilão em que foi feito o lance.
